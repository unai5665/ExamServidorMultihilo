# Servidor MultiHilo - Usuario y Contraseña

Este proyecto es una aplicación de chat en **Java** con arquitectura cliente-servidor. Permite que varios clientes se conecten a un servidor que deben introducir un usuario y una contraseña.

---

## 🛠️ Tecnologías utilizadas
- **Java SE**
- **Sockets TCP** para la comunicación cliente-servidor
- **Multithreading** para manejar múltiples clientes simultáneamente
